#  Retail Sales Data Analytics Project

##  Project Overview

This project focuses on analyzing retail sales data to uncover meaningful business insights related to customer behavior, product performance, sales trends, and profitability.

The goal was to simulate a real-world data analytics workflow  from raw data cleaning to business insight generation and dashboard creation.

---

##  Objectives

* Clean and preprocess raw retail datasets
* Perform exploratory data analysis (EDA)
* Answer key business questions using SQL
* Build an interactive dashboard for decision-making support

---

##  Tools & Technologies Used

* **Python** – Pandas, NumPy, Matplotlib, Seaborn
* **SQL Server** – Business query analysis
* **Power BI** – Dashboard creation & visualization
* **Excel/CSV datasets**

---

##  Analysis Performed

* Data cleaning and handling missing values
* Feature engineering and dataset preparation
* SQL queries to analyze revenue, returns, and customer patterns
* Visualization of sales trends and product performance
* Creation of an interactive Power BI dashboard

---

##  Key Business Insights

* The **Home & Kitchen** category showed higher return rates, affecting profitability
* **Personal Care** products generated strong profit margins
* **Online sales** underperformed compared to in-store sales, indicating growth potential
* Certain customer segments contribute disproportionately to revenue
